# pykitsl
 A robust package with efficients, fasts, and memory saver solutions to deal with different kinds of objects in python such as, strings, functions, decorators, methods, ranges, sequences, etc...

Un paquete robusto con soluciones eficientes, rápidas y que ahorran memoria para tratar con diferentes tipos de objetos en Python, como cadenas, funciones, decoradores, métodos, rangos, secuencias, etc.
